Paper #1
Does Reinforcement Learning Really Incentivize Reasoning Capacity in LLMs Beyond the Base Model?
F — Force
Force change
Unchanged
The reinforcement learning force used by RLVR is identical to existing RL. Only the reward type differs (verifiable rewards) — no new intrinsic force is created. F has not changed.
S — Scope
Scope change
Partially changed
The key finding is that RLVR actually contracts S (scope). When k is large, the Base model surpasses RLVR — proving S contraction. This is an observation of existing S, not creation of new S.
D — Dimension
Dimension change
Unchanged
The 2D matrix (Vertex structure) dimension is completely unchanged. No transition to Face structure. RLVR only adjusts probability distributions within the existing dimension. D is fully invariant.
T — Structural Transfer
T = 0
No transfer of new structural characteristics. RLVR only adjusts efficiency within the Base Model's structure — it cannot create new reasoning structures. Only Distillation achieves T=1.

Structurology verdict
Analysis paper — Insight without structural innovation
F and D are unchanged; only the S contraction phenomenon was newly discovered. This is an analysis paper without structural innovation. However, empirically identifying why RLVR has limitations provides important insight. From a Structurology perspective, this paper discovered a structural flaw (S contraction) in RLVR, but did not propose a new structure to resolve it.
Structural interpretation
RLVR operates by leaving F and D unchanged while narrowing S (scope). From Structurology's core proposition — "structure determines characteristics" — since the structure was not changed, fundamentally new characteristics cannot emerge. This also explains why 6 different RL algorithms all show similar results. Even with different algorithms, if the FSD structure is the same, the outcome is the same. The reason Distillation is different: T=1 — the teacher model's structure is actually transferred to the student model.
Paper #2
Quiet-STaR: Language Models Can Teach Themselves to Think Before Speaking
F — Force
Force change
Newly created
A new force is created — generating intrinsic rationales at every token position. Where STaR derived force only from QA correct answers, Quiet-STaR derives force from arbitrary text prediction itself. F has been newly created.
S — Scope
Scope change
Dramatically expanded
Where STaR's scope was limited to QA datasets, Quiet-STaR expands S to all internet text. "Language modeling itself is the arena for reasoning learning" — a paradigm shift. S has been dramatically expanded.
D — Dimension
Dimension change
Partially expanded
A new dimension is added — separating the Think (internal reasoning) and Talk (output) layers. The Mixing Head connects these two dimensions. Not a complete Face structure, but D has been partially expanded by adding a reasoning layer above the 2D matrix.
T — Structural Transfer
T = 1
The new structure (intrinsic reasoning layer) actually transfers into zero-shot performance gains. GSM8K 5.9%→10.9%, CommonsenseQA 36.3%→47.2%. New characteristics emerged without fine-tuning.

Structurology verdict
Structural innovation — F, S, and D all changed
F (new intrinsic reasoning force), S (expanded to all text), D (think/talk layer added) — all three axes changed. T=1 transfer occurred, with new structure generating new characteristics. This paper precisely embodies Structurology's proposition: "structure determines characteristics." The remaining challenge is that D achieved only partial expansion rather than a complete Face structure.
Structural interpretation
Quiet-STaR is an attempt to transplant the human cognitive structure of "think before speaking" into a language model. From a Structurology perspective — it adds a new dimension (internal reasoning) on top of the existing 1D token sequence, and gives that dimension F (force). Most innovatively, S is expanded from QA to arbitrary text. The insight that "all text already contains implicit reasoning" grounds this S expansion. From an Ω (emergence) perspective — the Think layer, Talk layer, and original text form three relationships that meet through the Mixing Head, and it is precisely from this that new characteristics (zero-shot reasoning ability) emerge. This aligns exactly with Structurology's axiom: 3 or more relations create structure.
