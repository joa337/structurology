분석 논문 #1
Does Reinforcement Learning Really Incentivize Reasoning Capacity in LLMs Beyond the Base Model?
F — Force · 힘
힘의 변화
기존 동일
RLVR이 사용하는 강화학습의 힘 자체는 기존 RL과 동일합니다. 보상 신호의 종류(검증 가능한 보상)가 다를 뿐, 새로운 내재적 힘을 만들어내지 않습니다. F는 변하지 않았습니다.
S — Scope · 범위
범위의 변화
부분 변화
이 논문의 핵심 발견은 RLVR이 S(범위)를 오히려 축소시킨다는 것입니다. pass@k가 클 때 Base가 RLVR을 앞서는 현상이 S 축소를 증명합니다. 새로운 S를 만든 것이 아니라 기존 S의 변화를 관찰했습니다.
D — Dimension · 차원
차원의 변화
기존 동일
2D 행렬(Vertex 구조) 기반의 차원은 전혀 변하지 않았습니다. Face 구조로의 전환은 없으며, RLVR은 기존 차원 안에서 확률 분포만 조정합니다. D는 완전히 불변입니다.
T — Transfer · 사상 전이
T = 0
새로운 구조적 특성의 전이가 일어나지 않았습니다. RLVR은 Base Model의 구조 안에서 효율만 조정할 뿐, 새로운 추론 구조를 만들지 못합니다. Distillation만 T=1입니다.

구조학 최종 판정
분석 논문 — 혁신 없음, 통찰 있음
F·D는 변하지 않았고 S의 축소 현상만 새롭게 발견했습니다. 구조적 혁신이 없는 분석 논문입니다. 그러나 "왜 RLVR이 한계를 가지는가"를 실증적으로 규명한 점에서 중요한 통찰을 제공합니다. 구조학적으로 보면 이 논문은 기존 구조(RLVR)의 S 축소라는 구조적 결함을 발견했지만, 그것을 해결할 새로운 구조를 제시하지는 못했습니다.
구조학적 해석
RLVR은 F·D를 그대로 두고 S(범위)만 좁히는 방식으로 작동합니다. 이는 구조학의 핵심 명제 "구조가 특성을 결정한다"에서 볼 때, 구조를 바꾸지 않았으므로 근본적으로 새로운 특성이 창발할 수 없습니다. 6개의 서로 다른 RL 알고리즘이 모두 유사한 결과를 보인 것도 이 때문입니다. 알고리즘이 달라도 FSD 구조가 같으면 결과는 같습니다. Distillation이 다른 이유는 T=1 — 교사 모델의 구조가 실제로 학생 모델로 전이되기 때문입니다.
분석 논문 #2
Quiet-STaR: Language Models Can Teach Themselves to Think Before Speaking
F — Force · 힘
힘의 변화
신규 창출
모든 토큰 위치에서 내재적 추론(rationale)을 생성하는 새로운 힘을 만들었습니다. 기존 STaR이 QA 정답에서만 힘을 얻었다면, Quiet-STaR은 임의 텍스트 예측 자체에서 힘을 얻습니다. F가 새롭게 창출되었습니다.
S — Scope · 범위
범위의 변화
획기적 확장
STaR의 범위가 QA 데이터셋에 한정됐다면, Quiet-STaR은 인터넷 텍스트 전체로 S를 확장했습니다. "언어 모델링 자체가 추론 학습의 장"이라는 패러다임 전환입니다. S가 획기적으로 확장되었습니다.
D — Dimension · 차원
차원의 변화
부분 확장
Think(내부 추론)와 Talk(출력) 레이어를 분리하는 새로운 차원을 추가했습니다. Mixing Head가 두 차원을 연결합니다. 완전한 Face 구조는 아니지만 2D 행렬 위에 추론 레이어를 얹는 방향으로 D가 부분 확장되었습니다.
T — Transfer · 사상 전이
T = 1
새로운 구조(내재적 추론 레이어)가 실제로 zero-shot 성능 향상으로 전이됩니다. GSM8K 5.9%→10.9%, CommonsenseQA 36.3%→47.2%. 파인튜닝 없이도 새 특성이 창발했습니다.

구조학 최종 판정
구조적 혁신 논문 — F·S·D 모두 변화
F(내재적 추론 힘 신규), S(전체 텍스트로 확장), D(think/talk 레이어 추가) — 세 축이 모두 변화했습니다. T=1 전이가 일어나 새로운 구조가 새로운 특성을 만들어냈습니다. 구조학의 명제 "구조가 특성을 결정한다"를 정확히 구현한 논문입니다. 다만 D가 완전한 Face 구조가 아닌 부분 확장에 그친 점은 향후 과제입니다.
구조학적 해석
Quiet-STaR은 "말하기 전에 생각하라"는 인간의 인지 구조를 언어 모델에 이식하려는 시도입니다. 구조학적으로 보면 — 기존 1D 토큰 시퀀스 위에 내부 추론이라는 새로운 차원을 추가하고, 그 차원이 F(힘)를 갖게 한 것입니다. 특히 S를 QA에서 임의 텍스트로 확장한 것이 가장 혁신적입니다. "모든 텍스트에 이미 추론이 내재되어 있다"는 통찰이 S 확장의 근거입니다. Ω(창발) 관점에서 — Think 레이어와 Talk 레이어와 원본 텍스트라는 세 관계가 Mixing Head를 통해 만나면서 비로소 새로운 특성(zero-shot 추론 능력)이 창발합니다. 3의 관계가 구조를 만든다는 구조학의 공리와 정확히 일치합니다.
